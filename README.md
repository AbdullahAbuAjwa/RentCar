# RentCarProject

* **This project contains 5 pages**
    1. **Home page**
    2. **All cars page**
    3. **Filter page**
    4. **Rent info page**

# How to run the project

* Open project folder.
* Open your `cmd.exe`
* Type: `npm install`
* Then type `npm run start`
* Then go to `localhost:1234/`
